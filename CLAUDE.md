# Impax Asset Management - Homepage

## Overview

Proof-of-concept homepage for Impax Asset Management (impaxam.com), featuring a responsive masonry layout for latest insights with card flip animations on desktop and carousel on mobile.

## Commands

```bash
# Run (simple local server)
python3 -m http.server 8000
# Then open http://localhost:8000

# Alternative with Node
npx serve .
```

## Project Structure

```
/
├── index.html              # Main homepage
├── src/
│   ├── css/
│   │   └── styles.css      # All styles (responsive, animations)
│   ├── js/
│   │   └── main.js         # Carousel, interactions, scroll effects
│   └── assets/             # Images, icons (placeholder)
└── public/                 # Static assets (placeholder)
```
